# Build Your APK in 3 Simple Steps (No Installation Required!)

## Step 1: Push to GitHub

```bash
cd /Users/akashdwivedi/Desktop/FunProjects/recipeCam

git add .
git commit -m "Initial commit"

# Create a new repo on github.com, then:
git remote add origin https://github.com/YOUR_USERNAME/recipecam.git
git push -u origin main
```

## Step 2: Wait for Build

- Go to your GitHub repo
- Click "Actions" tab
- Watch the build run (takes ~5-10 minutes)

## Step 3: Download APK

- Once build completes, click on it
- Scroll down to "Artifacts" section
- Download "RecipeCam-APK"
- Unzip and install on Android!
